Flask REST API
==============

CRUD REST API на Flask поверх базы данных на MySQL.

Установка и запуск
------------------
```bash
git clone https://github.com/cochegarka/eduex-flask-api.git
# ... настройка виртуального окружения
# ... настройка .env
./run.sh
```

Пример .env
-----------
```
DB_HOST=127.0.0.1
DB_NAME=eduex
DB_USER=eduex_demander
DB_PASS=123456
PORT=3002
SPA_URL=http://localhost:5000
```
